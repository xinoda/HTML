Flex Box

Flexbox é um mecanismo do CSS3 que permite aplicar facilmente alinhamento e distribuição de elementos e conteúdos no HTML.

Na prática, é uma técnica diferente para solucionar problemas e minimizar o grande uso de paddings, margins, floats, positions na resolução de dificuldades na construção de layouts.

O módulo de layout de caixa flexível, facilita o projeto estrutura de layout responsivo flexível sem usar flutuar ou posicionamento.

Para começar a usar o modelo Flexbox, você precisa primeiro definir um contêiner flexível.

Um layout flexível deve ter um elemento pai com a propriedade display definida como flex. Os elementos filhos diretos do contêiner flexível tornam-se automaticamente itens flexíveis.

As propriedades do flexbox são os seguintes:

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

flex-direction

A propriedade define em que direção o contêiner quer empilhar os itens flex.

Seus valores são:

  • column: empilha os itens flexíveis verticalmente (de cima para baixo)
  • column-reverse: empilha os itens flexíveis verticalmente (mas de baixo para cima)
  • row: empilha os itens flexíveis horizontalmente (da esquerda para a direita)

  • row-reverse: empilha os itens flexíveis horizontalmente (mas da direita para a esquerda)

flex-wrap

A propriedade especifica se os itens flexíveis devem ser ou não, de acordo com a viewport.

Seus valores são:

  • wrap: especifica que os itens flexíveis serão enrolados, se necessário
  • nowrap: especifica que os itens flexíveis não serão enrolados (este é o padrão)
  • wrap-reverse: especifica que os itens flexíveis irão embrulhar se necessário, em ordem inversa

justify-content

propriedade é usada para Alinhe os itens flex.

Seus valores são:

  • center:O valor alinha os itens flexíveis no centro do contêiner
  • flex-start: O valor alinha os itens flexíveis no início do recipiente (Isto é o padrão)
  • flex-end: O valor alinha os itens flexíveis no final do recipiente
  • space-around: o valor exibe os itens flexíveis com espaço antes, entre, e depois das linhas
  • space-between: O valor exibe os itens flexíveis com espaço entre o Linhas de linha

align-items

A propriedade é usada para alinhar os itens flex.

Seus valores são:

  • center:O valor alinha os itens flexíveis no meio do contêiner
  • flex-start: O valor alinha os itens flexíveis na parte superior do recipiente
  • flex-end: O valor alinha os itens flexíveis na parte inferior do recipiente
  • stretch: O valor estende os itens flexíveis para encher o recipiente (Isto é o padrão)
  • baseline: O valor alinha os itens flexíveis, como suas linhas de base, alinham-se

align-content

A propriedade é usada para alinhar as linhas flexíveis.

Seus valores são:

  • space-between:O valor exibe as linhas flexíveis com o mesmo espaço entre elas
  • space-around: valor exibe as linhas flexíveis com espaço antes, entre, e depois deles
  • stretch: O valor estica as linhas flexíveis para assumir o restante espaço (isso é padrão)
  • center: O valor mostra as linhas flexíveis no meio do recipiente
  • flex-start: O valor exibe as linhas flexíveis no início do recipiente
  • flex-end: O valor exibe as linhas flexíveis no final do recipiente
