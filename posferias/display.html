Valor do bloco em linha

Cada elemento HTML tem um valor de exibição padrão, dependendo do tipo de elemento é que é. Os dois valores de exibição mais comuns são bloco e em linha.

Elementos de nível de bloco

Um elemento de nível de bloco sempre começa em uma nova linha e os navegadores automaticamente adicionar algum espaço (uma margem) antes e depois do elemento.

Um elemento de nível de bloco ocupa sempre a largura total disponível (estoque para a esquerda e para a direita, tanto quanto pode).

Elemntos linha

Um elemento inline não começa em uma nova linha. Um elemento em linha ocupa apenas a largura necessária.

Comparado com display: inline, o principal A diferença é que display: inline-block permite para definir uma largura e altura no elemento.

Além disso, com display: inline-block, as margens superior e inferior/respectivamente são respeitados, Mas com o display: inline não.

Comparado com display: block, a principal diferença é que display: inline-block não adiciona uma quebra de linha após o elemento, para que o elemento possa estar ao lado de outros Elementos.
